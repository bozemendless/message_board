# 留言板 Message_Board

Message_Board 是一個以 Django 撰寫而成的動態留言板。

作品連結：https://shichenx.com/messages/

![shichenx com_messages_](https://user-images.githubusercontent.com/98375601/212478046-38c22e78-19ec-471f-88c0-55918dac7c81.png)

## 使用技術
### 基礎設施
 -  Docker
### 環境
 - Linux / Ubuntu
 - Python / Django
### 資料庫
 - MySQL (AWS RDS)
### AWS Services
 - EC2
 - S3
 - CloudFront
 - RDS (MySQL)
### 網路
 - HTTP / HTTPS
 - DNS
 - NGINX
 - SSL
### 版本控制
 -  Git / GitHub
### 架構
 - MVT Pattern
 - Restful API
### Frontend
 - HTML
 - CSS
 - JavaScript
 - AJAX
